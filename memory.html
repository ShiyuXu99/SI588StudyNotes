Memories

#SI588

Working Memory

Short-term memory - Temporary buffer for information, what they used to call working memory.

Definition: A temp buffer for information that’s currently in used.

Central executive

  • Control three different sub system
  • decides where to focus attention, does not store information.

Visuospatial sketchpad

  • temporary store for visual/spatial information
    • ex. When you look at a map (what that map looks like)

Phonological loop

  • temporary store for auditory and verbal information
    • Sound, music

Episodic Buffer

  • Orders, sequences and events to communicate to long-term memory, this is why patterned things are easier to remember.

Characteristic of Working Memory

  • Capacity is very small. Around 4+- 1 things that we can remember, Age can affect this number
  • Fades very fast
    • if we want to maintain this memory, we have to constant refresh, and keep items in WM
    • ex. A phone number to memorize, you will constantly repeat the number, know as maintenance rehearsal

Things affecting working memory

  • Numbers of items need to remember
    • chunking can increase recall (ex. Separate things into three characters)
  • Similarity reduces recall
  • Repetition can lead to false recall
  • Interruption will decrease working memory

Long-Term Memory

  • Where we store all our memories.

Procedure memory

  • Memories that are hard to describe( riding a bike, tie a shoe)

Declarative Memory

Semantic - Factual memory, such as meaning of words, thing you know about different topics

Episodic Memory - Memory of different events.

Memory have “colors”, emotions are attached to the memories.

Learning

Moving working memory to long-term memory.

Something learned previously can influence your ability to learn something new:

  • Positive transfer: Previous learning HELPS learn something new. (ex. Drive a golf car help you drive a real car in the future)
  • Negative transfer: Previous learning INHIBITS your ability to learn something new.

Long-term memory retrieval

Recognition

  • realizing that a stimulus matches a goal
  • This is the easier method (like a multiple choice question)

Recall

  • remembering the right option with no stimulus
  • factors that can help you remember ( routines and memorization tricks, context and state are also factors)
  • When in the same emotion, memory is easier to recall.

Proactive interference & Retroactive interference - old memory are replaced by the new one.

Long term memory lose strength, so it’s harder to retrieve.

Factors affect retrieval of LTM (long term memory)

  • Strength of item in LTM
    • effect by frequency and recency
  • Numbers of association between item and other things

Habituation

  • Built through repetition in a consistent context over time, with embedded rewards
  • No repetition, no habit.
  • No consistency, slower habit formation.

Organization of Memory

Four types of organization

Semantic networks

  • How things are linked together

Schemas

  • Mental concept that help organize concepts.
  • ex. Dogs have tails, they are furry
  • if something seems out of place, that’s because schemas are conflicting with
  • Script - special type of schema

Mental models

  • Theories hold of how things works
  • planning and trouble shooting

Cognitive maps

  • Locations and spacial environment.
  • ex. Thinking where the classroom are in a building, where the shopping cart is in an online website

Modern view of Memory

  • There is no physiological difference between WM and LTM
  • Perception activates a neural pattern
  • All memories are patterns of neural activity
  • Attention can focus on only a few patterns at a time
  • All observations from “classic” view still hold

Implementation in Design

Designs needs to :

  • Minimize working memory’s load
  • Provide redundant visual echoes with sound info
    • Hold user hands and provide place holders through sequential tasks so they don’t lose their place
  • Exploit chunking (physical size, meaningful sequences, letters vs. numbers, and separation of letters and numbers)
  • Minimize confusability
  • Avoid unnecessary zeroes in codes
  • Keep WM in mind when developing instructions
    • Ensure congruence
  • Avoid negative phrasing

Design systems to:

  • Encourage regular use (increase frequency and recency)
  • Encourage active verbalization reproduction of info
  • Standardize
  • Use memory aides
  • Design info to be remembered
  • Design helpful habits
  • Support development of correct mental models

Example:

  • Shopping site memorize what you usually buy so you don’t have to remember daily things.
  • Visual cues for modes and sequence (controller that can’t separate modes)
